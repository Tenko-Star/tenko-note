需要修改两个文件，分别是`tsconfig.json`和`vite.config.ts`

其中，`tsconfig.json修改如下`

```json
{
  "compilerOptions": {
    // ...其他配置
    "paths": {
      "@/*": ["./src/*"] // 配置路径别名
    }
  }
}
```

`vite.config.ts`修改如下

```typescript
export default defineConfig(async () => ({
    // ...其他配置
    resolve: {
        // 配置路径别名
        alias: {
            '@': path.resolve(__dirname, './src'),
        },
    },
}))
```
